Jennifer Chapman Photography

jchapmanphotography.com is a website I created displaying the photographic work of Jennifer Chapman. As a world class photographer Jennifer came to me with the urge to create a web portfolio consisting of her numerous photographic projects. Each project consists of multiple images and each project consists of a description.

The challenge of creating a dynamic and easy to use website was solved by creating a custom CMS for Jennifer. The CMS I created for this project was greatly inspired by Stacy. Creating a custom framework was very beneficial because  I could taylor make it for a photography gallery, unlike most text based CMS which are designed with blogging in mind.

One main difference between this framework and most is that the user can upload files via an FTP client and not be troubled by managing an admin dashboard. The ease of dragging and dropping files in and out of an FTP client provides fast and bulk uploads and downloads than using a traditional admin dashboard.  Another chief added advantage by using the file-system to manipulate page content is that their is no database management, which means a site is easy to deploy, relocate, and download.
